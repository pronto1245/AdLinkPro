ТЗ: Страница «Дашборд / Главная» — кабинет Партнёра
🎯 Цель:
Предоставить партнёру (арбитражнику, вебмастеру) быструю сводку по эффективности его трафика, статусу офферов, доходам и активности постбеков.

📌 Основные элементы интерфейса:
1. Карточки-метрики (основные показатели за выбранный период):
💰 Общий доход (за выбранный период)

🎯 Конверсии (общий лид/рег/деп)

👥 Клики / Уники

💵 EPC (заработок за клик)

⚖️ CR % (конверсия)

📦 Активные офферы

🔁 Постбеки получены / отправлены

⏳ Ожидаемый доход (pending)

✅ Подтверждённый доход

❌ Отклонённый доход

🕒 Среднее время удержания трафика

2. Графики:
Интерактивные, с фильтрацией по времени, типу действия, устройству и офферу.

📈 График дохода по дням

📉 График CR и EPC

📊 График лидов / депозитов / кликов

📍 График трафика по странам / гео

🔁 График активности постбеков

3. Таблица: ТОП офферы (по доходу / CR / кликам / EPC):
Оффер	Статус	Клики	CR %	Конверсии	Доход	EPC	Фрод %

4. Интерактивный фильтр:
📆 Временной диапазон: сегодня / вчера / неделя / месяц / произвольный

🌍 Гео

📱 Устройство / ОС

📦 Оффер

🔁 Тип действия: Клик / Рег / Деп / Лид

5. Уведомления / Задачи:
🔥 Новый оффер добавлен

⛔ Оффер отключен / приостановлен

💬 Сообщение от рекламодателя

⚠️ Обнаружена фрод-активность

🛠 Не задан постбек

6. Быстрые действия:
➕ Подключиться к новому офферу

⚙️ Настроить постбек

📤 Экспорт статистики

📊 Перейти в аналитику

📦 Перейти к офферам

7. Smart-алерты / Рекомендации (если включено):
📍 Подбор офферов по трафику (AI)

📢 Рекомендации по оптимизации кампаний

🔍 Предупреждение об аномалиях (высокий CTR, подозрительный трафик и т.д.)

8. Если партнёр с командой (Team):
Отображается блок:

👤 Список баеров / подаккаунтов (если хочет подключить свою команду, то сделать это подпиской-платная функция, запрос и оплата отправляется владельцу)

📈 Вклад каждого в доход

💼 Отдельные карточки по каждому участнику

🛠 Технические детали:
API: GET /affiliate/dashboard

Все данные фильтруются по user_id (партнёр)

Графики строятся на основе статистики, постбеков и офферов

Данные обновляются раз в 10–15 секунд при активном открытом дашборде (если нужно)
